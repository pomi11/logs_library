◆ connect() [1/2]

int DirectConnection::connect ( )
overridevirtual

umozliwia utworzenie polaczenia sieciowego, uzywa wczesniej ustawionych danych

jesli uzytkownik i/lub haslo nie zostaly wczesniej podane, a serwer ich wymaga, ustanawia polaczenie bez logowania

Returns
0 - polaczono i zalogowano
1 - podany uzytkownik zostal odrzucony przez serwer
2 - podany uzytkownik zostal zaakceptowany, serwer czeka na haslo
3 - podane haslo jest nieprawidlowe
4 - uzytkownik nie zostal podany
-1 - blad polaczenia
-2 - brak hosta
